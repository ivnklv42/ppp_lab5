# Описание проекта
Данное десктопное приложение представляет собой программу для управления онлайн магазином. Приложение построено на Java
с использованием JavaFX. Данные хранятся в SQLite. Представлена реализация как Java-проектом, так и исполнительным файлом (.exe), запакованным в 
.msi установщик при помощи jpackage для запуска на любом пк с Windows, не имеющем JRE.

# Функциональные возможности
## Авторизация:
- Проверка на нахождение пользователя в зарегистрированных
## Управление списком товаров:
- Добавление нового товара
- Редактирование существующего товара
- Удаление существующего товара
- Отображение всех существующих товаров
- Замена данных о товарах из файла
- Обновление данных о товарах из файла
- Выгрузка данных о товарах в файл 
## Управление списком покупателей:
- Добавление нового покупателя
- Добавление нового заказа для существующего покупателя
- Удаление существующего покупателя
- Отображение всех существующих покупателей с их заказами
- Замена данных о покупателях из файла
- Обновление данных о покупателях из файла
- Выгрузка данных о покупателях в файл
